- <span data-ttu-id="75080-106">[.nuspec 資訊清單](../reference/nuspec.md)：套件可以根據專案的目標 Framework，指出專案要包含的確切套件。</span><span class="sxs-lookup"><span data-stu-id="75080-106">[.nuspec manifest](../reference/nuspec.md): A package can indicate distinct packages to be included in a project depending on the project's target framework.</span></span>
