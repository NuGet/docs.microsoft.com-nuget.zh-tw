<span data-ttu-id="7b727-229">[.NET Standard](/dotnet/standard/net-standard)可簡化二進位相容架構之間的參考, 讓單一目標架構能夠參考其他專案的組合。</span><span class="sxs-lookup"><span data-stu-id="7b727-229">[.NET Standard](/dotnet/standard/net-standard) simplifies references between binary-compatible frameworks, allowing a single target framework to reference a combination of others.</span></span> <span data-ttu-id="7b727-230">(背景請參閱 [.NET 入門](/dotnet/articles/standard/index)。)</span><span class="sxs-lookup"><span data-stu-id="7b727-230">(For background, see the [.NET Primer](/dotnet/articles/standard/index).)</span></span>
