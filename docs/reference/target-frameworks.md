<span data-ttu-id="95aba-232">[.NET Standard](/dotnet/standard/net-standard) 可簡化二進位相容架構之間的參考，讓單一目標架構參考其他人的組合。</span><span class="sxs-lookup"><span data-stu-id="95aba-232">[.NET Standard](/dotnet/standard/net-standard) simplifies references between binary-compatible frameworks, allowing a single target framework to reference a combination of others.</span></span> <span data-ttu-id="95aba-233">(背景請參閱 [.NET 入門](/dotnet/articles/standard/index)。)</span><span class="sxs-lookup"><span data-stu-id="95aba-233">(For background, see the [.NET Primer](/dotnet/articles/standard/index).)</span></span>
