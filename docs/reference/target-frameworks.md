<span data-ttu-id="d8109-104">NuGet 在各種位置使用目標 Framework 參考，具體識別並隔離套件的 Framework 相依元件：</span><span class="sxs-lookup"><span data-stu-id="d8109-104">NuGet uses target framework references in a variety of places to specifically identify and isolate framework-dependent components of a package:</span></span>

- <span data-ttu-id="d8109-106">[.nuspec 資訊清單](../reference/nuspec.md)：套件可以根據專案的目標 Framework，指出專案要包含的確切套件。</span><span class="sxs-lookup"><span data-stu-id="d8109-106">[.nuspec manifest](../reference/nuspec.md): A package can indicate distinct packages to be included in a project depending on the project's target framework.</span></span>
