# <a name="nuget-warning-nu3035"></a>NuGet 警告 NU3035

### <a name="issue"></a>問題

完整的憑證鏈結無法建立存放庫副署的簽章憑證。


### <a name="solution"></a>方案

連絡的時間戳記授權單位，以找出問題的來源。
