# <a name="nuget-error-nu5008"></a>NuGet 錯誤 NU5008
<pre>Manifest file not found at 'F:\project\project.nuspec'</pre>

### <a name="issue"></a>問題

傳遞至 NuGet 套件作業的 nuspec 檔案不存在。


### <a name="solution"></a>解決方案

請確定磁片上已有傳遞至 NuGet 套件命令的 nuspec 檔案。
