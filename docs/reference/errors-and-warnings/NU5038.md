# <a name="nuget-error-nu5038"></a>NuGet 錯誤 NU5038
<pre>The readme file 'readme.txt' has an invalid extension. It must end in .md.</pre>

### <a name="issue"></a>問題

讀我檔案必須是 markdown 檔案，且必須以 '*md*' 結尾。

### <a name="solution"></a>解決方法

針對讀我檔案使用 markdown 副檔名。
