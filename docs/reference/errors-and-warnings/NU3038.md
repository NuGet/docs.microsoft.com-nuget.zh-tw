# <a name="nuget-warning-nu3038"></a>NuGet 警告 NU3038

### <a name="issue"></a>問題

驗證設定需要存放庫副署，但封裝無需儲存機制的副署。


### <a name="solution"></a>方案

使用具有副署的儲存機制封裝的封裝來源。
