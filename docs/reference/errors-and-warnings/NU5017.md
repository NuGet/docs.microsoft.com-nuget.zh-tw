# <a name="nuget-error-nu5017"></a>NuGet 錯誤 NU5017
<pre>Cannot create a package that has no dependencies nor content.</pre>

### <a name="issue"></a>問題

所儲存的封裝不包含任何封裝參考、元件或架構參考。


### <a name="solution"></a>解決方案

請確定所建立的封裝包含元件或封裝參考。
