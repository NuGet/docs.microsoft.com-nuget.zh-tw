# <a name="nuget-error-nu5013"></a>NuGet 錯誤 NU5013
<pre>Failed to build 'project.csproj'</pre>

### <a name="issue"></a>問題

使用選項執行 NuGet 套件作業時，無法建立專案 `-Build` 。


### <a name="solution"></a>解決方案

請修正專案中的組建錯誤，然後再試一次。 失敗的原因應該會顯示在主控台記錄中。
