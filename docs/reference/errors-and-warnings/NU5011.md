# <a name="nuget-error-nu5011"></a>NuGet 錯誤 NU5011
<pre>Unable to extract metadata from 'project.csproj'.</pre>

### <a name="issue"></a>問題

從元件或專案檔解壓縮中繼資料時，發生執行時間例外狀況。


### <a name="solution"></a>解決方案

請在上建立問題 https://github.com/NuGet/Home/issues 。
