# <a name="nuget-warning-nu3031"></a>NuGet 警告 NU3031

<pre>Package 'SamplePackage v1.0.0' from source 'https://contoso.com/index.json': The repository countersignature is invalid.</pre>

### <a name="issue"></a>問題

存放庫副署不正確。


### <a name="solution"></a>方案

請連絡副署封裝的儲存機制。
