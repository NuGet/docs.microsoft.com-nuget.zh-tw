# <a name="nuget-error-nu5047"></a>NuGet 錯誤 NU5047

<pre>The icon file size must not exceed 1 megabyte.</pre>


### <a name="issue"></a>問題 

指定為封裝圖示的檔案大於 1 mb。 NuGet 只允許檔案大小小於 1 MB 的圖示。


### <a name="solution"></a>方案

使用「影像編輯器」程式來減少封裝圖示檔的大小。
