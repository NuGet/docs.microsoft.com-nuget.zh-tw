# <a name="nuget-error-nu5002"></a>NuGet 錯誤 NU5002
<pre>Please specify a nuspec, project.json, or project file to use.</pre>

### <a name="issue"></a>問題

未提供 NuGet 套件作業的輸入檔。


### <a name="solution"></a>解決方案

請使用 nuspec、project.json 或 project 檔案的路徑，將套件命令當作引數。
