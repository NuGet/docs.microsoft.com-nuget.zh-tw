<span data-ttu-id="037d7-126">在 Visual Studio 15.8.4 和 NuGet.exe 4.8.1 的更新中，我們引進了可切換撤銷檢查模式的環境變數。</span><span class="sxs-lookup"><span data-stu-id="037d7-126">Update to Visual Studio 15.8.4 and NuGet.exe 4.8.1 where we introduced an environment variable to switch the revocation check mode.</span></span>
