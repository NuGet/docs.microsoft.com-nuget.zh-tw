## <a name="search-syntax"></a><span data-ttu-id="dd4c8-198">搜尋語法</span><span class="sxs-lookup"><span data-stu-id="dd4c8-198">Search Syntax</span></span>

<span data-ttu-id="dd4c8-199">在 nuget.org 上、從 NuGet CLI 中，以及 Visual Studio 的 NuGet 套件管理員延伸模組內，NuGet 套件搜尋的運作都相同。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-199">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="dd4c8-200">一般而言，會將搜尋套用至關鍵字和套件描述。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-200">In general, search is applied to keywords as well as package descriptions.</span></span>

- <span data-ttu-id="dd4c8-201">**關鍵字**：搜尋會尋找包含任何所提供關鍵字的相關套件。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-201">**Keywords**: Search looks for relevant packages that contain any of the provided keywords.</span></span> <span data-ttu-id="dd4c8-202">範例：`modern UI`。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-202">Example: `modern UI`.</span></span> <span data-ttu-id="dd4c8-203">若要搜尋包含所有已提供關鍵字的套件，請在詞彙之間使用 "+"，例如 `modern+UI`。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-203">To search for packages that contain all of the provided keywords, use "+" between the terms, such as `modern+UI`.</span></span>
- <span data-ttu-id="dd4c8-204">**詞組**：輸入以雙引號括住的字詞會尋找與這些字詞完全相同且不區分大小寫的相符項。</span><span class="sxs-lookup"><span data-stu-id="dd4c8-204">**Phrases**: Entering terms within quotation marks looks for exact case-insensitive matches to those terms.</span></span> <span data-ttu-id="dd4c8-205">範例：`"modern UI" package`</span><span class="sxs-lookup"><span data-stu-id="dd4c8-205">Example: `"modern UI" package`</span></span>

    ```

    ID:jQuery
    title:jquery                 # Searches title as shown on the package listing
    PackageId:jquery             # Match the package id exactly
    id:jquery id:ui              # Search for multiple terms in the id
    id:jquery tags:validation    # Search multiple properties
    id:"jquery.ui"               # Phrase search
    invalid:jquery ui            # Unsupported properties are ignored, so this

    ```
